# 🌅 Get a Wriggle On!

A simple, fun morning routine tracker for kids — built to run in the browser on a family iPad.

Each child gets their own kanban-style board to tick off their morning tasks and earn a little celebration when everything's done.

---

## Features

- **Two swimlanes** — one per child, each fully independent
- **Kanban board** — tasks move from To Do → Done with a tap
- **Customisable** — each child picks their own name and avatar emoji
- **Add & delete tasks** — build the right routine for your family
- **Daily auto-reset** — tasks reset fresh each morning automatically
- **Confetti & celebrations** — a dopamine hit every time a task is completed, and a "Mission Complete!" moment when the board is cleared
- **Progress bar** — so kids can see how close they are to done
- **iPad-optimised** — large touch targets, smooth animations, works offline via localStorage
- **Zero dependencies** — a single HTML file, no build tools, no server needed

---

## Getting started

Just open `index.html` in any browser. That's it.

To use on an iPad, AirDrop the file across and open in Safari — or host it anywhere (GitHub Pages, Netlify, etc.) and bookmark it on the home screen.
